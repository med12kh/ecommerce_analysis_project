# 🛒 E-commerce Data Analysis Project

## 📌 Description

This project analyzes an e-commerce dataset using Python (Pandas, NumPy, Matplotlib).

## ⚙️ Features

* Data cleaning (handling missing values, duplicates)
* Data transformation
* Revenue analysis (by category, product, date)
* Visualization (bar chart & line chart)
* Export cleaned data

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib

## 📊 Outputs

* Cleaned dataset (Excel)
* Revenue report (CSV)
* Charts

## ▶️ How to Run

```bash
pip install pandas numpy matplotlib openpyxl
python main.py
```
